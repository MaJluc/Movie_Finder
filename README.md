# 🎬 Movie Search Application
**Консольное приложение для поиска фильмов**

---

## 📌 Описание проекта

Интерактивное консольное приложение для поиска фильмов по базе данных **Sakila (MySQL)** с логированием поисковых запросов в **MongoDB**.  
Приложение позволяет пользователю быстро находить фильмы по ключевым словам, жанрам и диапазону годов выпуска, а также отслеживать популярные запросы.

---

## ⚙️ Функциональность

### 🔍 Поиск фильмов
- **По ключевому слову** (часть названия)  
  - Результаты ограничены 10 фильмами за раз  
  - Возможность просмотреть следующие 10 результатов
- **По жанру и диапазону годов**  
  - Перед поиском показывается список доступных жанров  
  - Отображается минимальный и максимальный год выпуска  
  - Пользователь может задать диапазон или конкретный год

### 🗂 Логирование
- Все поисковые запросы сохраняются в MongoDB

### 📊 Статистика
- Вывод 5 самых популярных запросов по частоте поиска  
- Вывод 5 последних запросов

### ⚠️ Обработка ошибок
- Корректная обработка ошибок подключения к базам данных  
- Проверка пользовательского ввода  
- Приложение не завершает работу аварийно

## 🛠 Технологический стек

- **Python 3.x**  
- **MySQL** — база данных с информацией о фильмах  
- **MongoDB** — логирование поисковых запросов  
- **Python-библиотеки:**  
  - `pymysql` — подключение к MySQL  
  - `pymongo` — работа с MongoDB  
  - Другие утилиты для форматирования и вывода в консоль

## 🛠 Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```env
# MySQL
MYSQL_HOST=host
MYSQL_USER=username
MYSQL_PASSWORD=password
MYSQL_DB=database_name

# MongoDB
MONGO_URI=Connection String
MONGO_DB=username
MONGO_COLLECTION=collection_name
```

> Важно: файл `.env` не добавляется на GitHub (см. `.gitignore`)
